### 仕様変更

1. [CODE VIEW] filename.py ... の表示をスティッキーヘッダーではなく、エディター領域に出したい。
2. 画面下部の 固定ステータスバーの情報エディター領域に出したい
3. ショートカットキーのヒントも画面上部に出したい
4. ショートカットキーのヒントは markdown でも出したい。
5. エディター領域の背景色を下記のように動的に変更する
   - Markdown / Light Mode: 明るい 暖色系
   - Markdown / Dark Mode: 暗い 暖色系
   - Code / Light Mode: 明るい 寒色系
   - Code / Dark Mode: 暗い 寒色系
6. Diagram Zoomについて、マウスカーソルが当たっているところを中心にズームしたい。
