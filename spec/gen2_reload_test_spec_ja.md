# GRSMD Gen2 Reload 機能 ユーザーテスト仕様書

**バージョン:** 2.0
**日付:** 2026-03-05
**対象:** `docs/index.html` (Vite ビルド出力)
**目的:** Reload ボタン、`r` キー、l/d テーマ切替 (コードビュー CSS-only)、keyHint 移設・自動非表示、2メッセージ toast 分岐、reload 時 preprocessInput 適用を確認する

## TC-R11: keyHint 自動非表示 (5秒タイムアウト)

| 項目         | 詳細                                                 |
| ------------ | ---------------------------------------------------- |
| **対象機能** | showKeyHint の 5秒タイマー自動非表示                 |
| **前提条件** | ページを新規に読み込み、初期状態であること           |
| **手順**     | 1. `.py` ファイルをページにドラッグ＆ドロップ        |
|              | 2. keyHint が表示されたら、何も操作せず待つ          |
| **期待結果** | - 約5秒後に keyHint がフェードアウトして非表示になる |

| 結果 | Pass / Fail / N/A                      |
| ---- | -------------------------------------- |
| 判定 | N/A                                    |
| 備考 | 問題なく消えたが、メッセージが不適切。 |

キーは列をそろせよ。
例：
[up][down] : scroll / [l] : light mode / [d] : dark mode
[r] : reload / [c] : clear / [n] : new tab
ただし、 : や / は業界標準に合わせたい。 標準的な書式を複数挙げて比較検討し、俺の判断を仰げ。
